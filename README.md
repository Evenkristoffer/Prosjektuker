# Prosjektuker

This project is a simple GTA VI wiki website made with plain HTML, CSS, and JavaScript.

## Files

- `index.html` is the main page.
- `leaks.html` is the leaks page.
- `media.html` is the media page.
- `styles.css` contains the styling.
- `script.js` contains the JavaScript for navigation, cards, and search.

## How To Run

Open `index.html` in a browser.

## Code Features

The JavaScript uses:

- arrays/lists
- `for` loops
- `if` statements
- simple functions
